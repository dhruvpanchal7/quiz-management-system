Quiz Management Plan

Assumptions, scope, and approach

#Goal
TO build a small production-ready quiz system, focusing on clean end to end flow rather than more feature which dont work as reliably.

#Assumptions
-Single admin user
-No login for quiz takers
-UI polishing is not a priority

#Scope
Admins : Login, create various types of quiz (MCQ, True/False, text, etc.)
Public Page: Take quiz by URL, View score after the submitting it

#Approach
-Implementing admin then quiz creation and quiz taking flow
-Verify all the systems to ensure they are working as intended

#Tech Stack
MongoDB Atlas
Tailwind CSS
Next.js(React + API routes)
Simple admin auth - cookie based
